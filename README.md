# Mountain View Commons — Scope of Work

Marketing proposal (Scope of Work) for Mountain View Commons. Prepared by Websquire, February 2026.

## Contents

- **index.html** — Main proposal page
- **styles.css** — Layout and typography
- **script.js** — Phase accordion toggles and keyboard accessibility

## View locally

Open `index.html` in a browser, or run a simple server:

```bash
# Python 3
python3 -m http.server 8000

# Then visit http://localhost:8000
```

## Deploy

Static files only. Works with GitHub Pages, Netlify, or any static host. Point the root or `index.html` as the entry.
